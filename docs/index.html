WebJS Documentation

Welcome to the official documentation for WebJS, a lightweight framework for building standard multipage web applications. It brings back the joy of web development with a simple, yet powerful set of utilities.

Registry

Persistent key-value storage that survives page reloads.

URL Handling

Utilities for parsing and manipulating URLs and query parameters.

Authentication

Helpers for managing user session and authentication tokens.

Event System

Lightweight publish/subscribe system for component communication.

Widget Loading

Dynamically load and execute widget content from external URLs.

Language

Simple methods for managing application language state.

Image Utilities

Load background images asynchronously with ease.
