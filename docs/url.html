URL Handling

WebJS provides a robust set of utilities for working with URLs, query parameters, and navigation.

$$.getUrl()

Returns the full current URL as a string.

const currentUrl = $$.getUrl();

$$.getUrlParam(name)

Retrieves the value of a specific query parameter. Returns null if not found.

// For URL: ?id=123&name=john
const id = $$.getUrlParam('id'); // "123"

$$.getUrlParams()

Returns all query parameters as a plain JavaScript object.

// For URL: ?category=books&sort=price
const params = $$.getUrlParams();
// { category: "books", sort: "price" }

$$.to(url, params, options)

Navigates to a new URL. It automatically handles base URL prepending for internal routes and can append query parameters.

// Navigate to internal route
$$.to('/dashboard');

// Navigate with parameters
$$.to('/search', { q: 'webjs' });

// Open in new tab
$$.to('https://google.com', {}, { target: '_blank' });
