Event System (Pub/Sub)

A lightweight messaging system that allows different parts of your application to communicate without being tightly coupled.

$$.subscribe(event, callback)

Listens for a specific event. Returns a subscription object that can be used to unsubscribe.

const sub = $$.subscribe('user.login', (data) => {
  console.log('User logged in:', data.name);
});

$$.publish(event, data)

Triggers an event and passes data to all subscribers.

$$.publish('user.login', { id: 42, name: 'Jules' });

subscription.unsubscribe()

Stops listening for the event.

const sub = $$.subscribe('ping', () => console.log('pong'));
// ... later
sub.unsubscribe();
