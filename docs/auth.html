Authentication

Manage user identity and session tokens easily with these helper methods.

$$.getAuthToken() / $$.setAuthToken(token)

Get or set the current authentication token (e.g., a JWT). The token is persisted in the Registry.

// Save token after login
$$.setAuthToken('abc.123.def');

// Retrieve token for API calls
const token = $$.getAuthToken();

$$.getAuthUser() / $$.setAuthUser(user)

Get or set the current user object. Useful for storing profile information like name, email, and roles.

$$.setAuthUser({
  id: 1,
  username: 'jules',
  role: 'admin'
});

const user = $$.getAuthUser();
console.log('Welcome back, ' + user.username);
