Widget Loading

Dynamically fetch and inject HTML content into your pages. WebJS automatically handles script execution within loaded content.

Usage

1. Define a container with the data-widget-url attribute:

<div data-widget-url="/api/widgets/sidebar">Loading...</div>

2. Initialize the widgets in your JavaScript:

document.addEventListener('DOMContentLoaded', () => {
  $$.loadWidgets();
});

How it works

When loadWidgets() is called, WebJS finds all elements with the data-widget-url attribute. It then performs a fetch request to each URL, injects the received HTML into the container, and identifies any <script> tags. These scripts are then executed in the context of the main page.
